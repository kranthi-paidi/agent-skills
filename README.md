# agent-skills

This is a public repository for agent skills used in the SRE Discipline.

It currently starts with a JMeter scripting skill for OpenAPI specifications, and it will continue to evolve as we add more tools and expand into additional SRE areas.

## skills

### openapi-to-jmeter

Turn an OpenAPI 3.x or Swagger 2.0 spec into a ready-to-run Apache JMeter test
plan, with correlation, assertions, a data-file template, and an audit of what
could and couldn't be wired automatically.

see [skills/openapi-to-jmeter/README.md](skills/openapi-to-jmeter/README.md)
